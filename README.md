# Customer Churn Prediction Using Machine Learning


## Project Overview

This project predicts whether a customer will leave a company using machine learning.


## Algorithms Used

- Logistic Regression
- Decision Tree
- Random Forest


## Features

- Data preprocessing
- Model training
- Accuracy evaluation
- Confusion Matrix
- ROC Curve visualization


## Technologies

Python
Pandas
Scikit-learn
Matplotlib
Seaborn


## How to Run


Install libraries:

pip install -r requirements.txt


Run notebook:

jupyter notebook


## Output

Model predicts customer churn probability.
